# Knot Agent

Private scaffold for initializing a local Knot Codex agent workspace.

Start Codex, use the `knot-setup` skill, and follow the install flow. After
installation, use `knot-workflow` to route knowledge, IM, attachments, and
deliverable tasks. The scaffold contains only templates and checks; component
repositories, runtime credentials, logs, customer data, and workspace files are
created locally.
